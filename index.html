    Installation et Configuration de GestSup

    Période : Du 06/09/2024 au 27/09/2024

    Mise en place de GestSup v3.2.50 sur un serveur Linux Debian 12, avec configuration réseau, sécurisation via HTTPS, et personnalisation de l'interface. Ajout du plugin Kanban pour une meilleure gestion visuelle des tickets.

    • Installation de GestSup v3.2.50 sur Linux Debian 12
    • Configuration réseau en IP fixe
    • Activation du HTTPS avec SSL sur Apache2
    • Personnalisation de l'accueil avec logo et photos d’équipe
    • Installation et paramétrage du plugin Kanban

    Compétences mobilisées :

    Administration Linux Apache2 Sécurité HTTPS/SSL Personnalisation UI GestSup
